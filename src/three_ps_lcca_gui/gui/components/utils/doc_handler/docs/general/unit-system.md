# Unit System

The measurement system applied to all dimensional inputs in this project.

## Options

| Option | Length | Mass | Volume |
|---|---|---|---|
| **SI (Metric)** | Metres (m) | Kilograms (kg) | Cubic metres (m³) |
| **Imperial** | Feet (ft) | Pounds (lb) | Cubic feet (ft³) |

## Important

> Changing the unit system after data entry **does not convert existing values** — it only updates the labels displayed. Re-enter values manually if you switch systems mid-project.
